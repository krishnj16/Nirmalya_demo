Step 1: Download and Install exiftool
•	Go to: https://exiftool.org/
•	Download and install it for your operating system.
Step 2: Run a Command to Extract GPS Data
Use the command line (Terminal or Command Prompt):
exiftool -csv -GPSLatitude -GPSLongitude -DateTimeOriginal -FileName path_to_photos > photos_gps.csv
Replace path_to_photos with the folder where your images are stored,This will create a CSV file (photos_gps.csv) with:
•	Filename
•	Latitude
•	Longitude
•	Date/time (if available)
